Differences between versions

The ready-made pdf is an easily printable and viewable version of The Tabula. Unfortunately, you cannot edit it.

The Tabula was originally written in LibreOffice Writer, using a significant amount of "advanced" techniques to get chapter headings and verse numbers working properly. Do NOT attempt to open the file in Microsoft Word, it WILL break the formatting. If you want to add to The Tabula, this is the file to use. Email me if you have any questions about formatting.

If you have any updates to the text, either message me on GitHub at WitnessTaube or email me so I can get it updated here.
